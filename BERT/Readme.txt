=======================================================
Negation and Uncertainty detection using BERT
=======================================================

1) The BERT.ipynb file requires Tensorflow 2.0 and Keras before being executed.

2) The directory libs contains scripts used with BERT.

3) Before using this model, the training corpus must be pre-processed using scripts from the Preprocessing directory. 
   We also provide a pre-processed version of the training data. This data can be download from:
   
   https://drive.google.com/drive/u/1/folders/1r7LO53y2YY42Z43YzR8E6PevUtrlM2vI
